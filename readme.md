# Instagram Downloader

Instagramの投稿（画像・動画）をダウンロードするためのシンプルなデスクトップアプリケーションです。

## 特徴

*   **簡単な操作**: URLを入力してボタンを押すだけでダウンロードできます。
*   **ログイン機能**: 必要に応じてInstagramアカウントにログインしてダウンロードできます（アクセス制限の回避などに有効）。
*   **自動フォルダオープン**: ダウンロード完了後、保存先のフォルダを自動的に開きます。
*   **マルチプラットフォーム**: Windows, macOS, Linuxに対応しています。

## 必要要件

*   Python 3.x
*   `instaloader` ライブラリ
*   `tkinter` (通常、Pythonの標準インストールに含まれています)

## インストール

1.  必要なライブラリをインストールします。

    ```bash
    pip install instaloader
    ```

## 使い方

1.  スクリプトを実行してアプリケーションを起動します。

    ```bash
    python instaloder.py
    ```

2.  **URL入力**: ダウンロードしたいInstagramの投稿URL（例: `https://www.instagram.com/p/xxxxxxx/`）を入力欄に貼り付けます。
3.  **ログイン設定 (任意)**:
    *   通常はログインなしで利用できます。
    *   頻繁なアクセスでブロックされる場合や、より安定した接続が必要な場合は、「ログインしてダウンロードする」にチェックを入れ、ユーザー名とパスワードを入力してください。
4.  **ダウンロード開始**: ボタンをクリックするとダウンロードが始まります。完了するとメッセージが表示され、保存フォルダが開きます。  
  ※ ダウンロードフォルダは自動的に生成されます。

## 注意事項

*   **アカウントのリスク**: 自動化ツールを使用したアクセスはInstagramの利用規約に抵触する可能性があります。過度な使用はアカウントの一時制限や凍結につながる恐れがあるため、自己責任でご利用ください（サブアカウントの利用を推奨します）。
*   **二段階認証**: 現在、二段階認証が有効なアカウントでのログインには対応していません。
*   **非公開アカウント**: ログインしていない状態では、非公開アカウント（鍵垢）の投稿はダウンロードできません。
